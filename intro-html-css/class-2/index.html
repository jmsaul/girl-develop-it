CSS: What is it?

CSS = Cascading Style Sheets

CSS is a "style sheet language" that lets you add style to the HTML elements on your page.

CSS works in conjunction with HTML, but it is not HTML itself.

CSS: What can it do?

All colored text, page layout, element positions, and sizes.

CSS: What does it look like?

The CSS "Rule"

The CSS "Rule":

selector {
  property: value;
}
  • A "block" of CSS code is a rule.
  • A rule starts with a selector.
  • A rule has sets of properties and values.
  • A property-value pair is a declaration.

CSS Syntax

The selector is how we choose an HTML element, or elements, to apply CSS declarations (properties and values) to.


selector {
  property: value;
}

CSS Syntax

Declarations contain the property and value of a CSS style you plan use on an HTML element, or elements.

  • Declarations always end with a semicolon.
  • Declaration groups are surrounded by curly brackets.

selector {
  property: value;
  property: value;
  property: value;
}

CSS Selectors

There are four types of CSS Selectors we will cover today:

  1. Element Selectors
  2. Id Selectors
  3. Class Selectors
  4. Descendant Selectors

Element Selector


Selects all paragraph elements.

p {
  property: value;
}

Selects all image elements.

img {
  property: value;
}

id Selector

Selects all HTML elements with an id of "footer".

#footer {
  property: value;
}

The associated HTML.

<p id="footer">Copyright 2011</p>

Class Selector

Selects all HTML elements with a class of "warning".

.warning {
  color: red;
}

The associated HTML.

<p class="warning">Run away!</p>

IDs vs. Classes

Ids - Should only apply to ONE unique element on a webpage. i.e., a webpage only has one footer.
  • The "#" is how you tell CSS "this is an id."

Class - Many elements can have the same class. i.e., there can be many warnings on one webpage.
  • The "." is how you tell CSS "this is a class name."

Descendant Selector

Selects all <em> elements that are inside <p> tags.

p em {
  color: yellow;
}

The associated HTML.

<p>This is <em>important.</em></p>

CSS Properties

Many CSS properties have self-explanatory names:

  • background-color
  • font-family
  • font-size
  • color
  • width
  • height


Comprehensive list of all CSS properties

Property: Color

The color property changes the color of text.

p {
  color: red;
}

div {
  color: #ff0000;
}

h2 {
  color: rgb(255, 0, 0);
}

Colors can be set via:

  • Color name
  • Hexadecimal value
  • RGB and RGBA values
  • ...and more.

For web-friendly color names, refer to the Standard X11 Color Scheme names here: Wikipedia: X11 Color Names

Property: Background-color

The background-color property changes the color of the background.


p {
  background-color: #000000;
}

Property: Font-family

The font-family property defines which font is used.


p {
  font-family: Arial;
}

You can declare font families in the following ways:

  • By specific font name
  • By generic font name
  • With a comma-separated list

Font Family - Multiple Values

With font-family, you can create a comma-separated list of different font names. This will create "fallbacks" that the browser can load in case a specific font style is unavailable.


p {
  font-family: "Helvetica", Arial, sans-serif;
}

Property: Font-size

The font-size property specifies the size of the font.

p {
  font-size: 14px;
}

h3 {
  font-size: 1.5em;
}

.warning {
  font-size: 80%;
}
  • Pixels - px. A set pixel value.
  • em - a relative unit of measure
  • Percentage - %

Property: Fonts (shorthand)

You can set more than one font style property at a time using the font property.

p {
  font-style: italic;
  font-weight: bold;
  font-size: 10px;
  font-family: sans-serif;
}
OR 
p {
  font: italic bold 10px sans-serif;
}

Full documentation on the font property can be found here: MDN: font CSS property.

Connecting CSS to HTML

3 ways to add CSS to your web site:

  1. "Inline"
  2. "Embedded"
  3. "External"

Inline CSS


	<p style="color:red;">Some text.</p>
  • CSS is added directly to an HTML element
  • Uses the style HTML attribute
  • Difficult to use and maintain in large projects
  • NOT preferred! Considered a bad practice.

Embedded CSS


<head>
  <style type="text/css">
    p {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>
  • CSS is written inside the HTML <head> element.
  • Uses the <style> tag.
  • The CSS rules only apply to the HTML document they are in.
  • Use with caution. Generally not a preferred method to add CSS to web sites.

Linked CSS Files


<head>
  <link rel="stylesheet" type="text/css" href="css/style.css" />
</head>
  • A separate CSS file is linked to a web page in the HTML <head> tag
  • Uses the <link /> tag
  • The CSS file can be a shared resource for many pages at a time
  • Reduces ﬁle size & bandwidth
  • Easy to scale and maintain in larger projects
  • Preferred by developers everywhere!

Let's develop it

  • Create a new CSS file in your HTML project folder called style.css
  • Add a <link /> to the file in the <head> of your HTML document
  • Add styles to change the colors, background colors, and fonts of different parts of your content
  • Try using ids and classes to change specific elements

Cascading

Styles "cascade" down. Order matters!

CSS:

p {
  color: blue;
  font-family: 'Helvetica', sans-serif;
}

.red {
  color: red;
}

.green {
  color: green;
}

#special {
  background: #000000;
}
HTML:

<h2>My Heading</h2>
<p class="green">Paragraph One</p>
<p class="red">Paragraph Two</p>
<p class="red" id="special">Paragraph Three</p>
